# Codex로 만든 TIL 사이트

## 오늘 배운 것

Codex를 사용해 GitHub Pages에 배포할 TIL 사이트를 만들었다. Markdown으로 기록하고, `posts.json`에 메타데이터를 관리하며, URL hash를 사용해 새로고침 후에도 현재 문서 위치를 유지하도록 구현했다.

## 핵심 정리

- Markdown 문서는 카테고리 폴더에 저장한다.
- 문서 메타데이터는 `posts.json`에서 관리한다.
- URL hash를 사용하면 새로고침 후에도 현재 문서 위치를 유지할 수 있다.

```js
window.addEventListener("hashchange", () => {
  const path = location.hash.replace(/^#/, "");
  console.log("현재 문서:", decodeURIComponent(path));
});
```

> 복잡한 빌드 도구 없이도 기록을 읽기 좋게 보여주는 구조를 만들 수 있다.
